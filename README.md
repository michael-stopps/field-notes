# field-notes
A lightweight World of Warcraft interface addon designed to enable quick note taking and add transcription functionality to records around the world.

## Features

* **Easy Note-taking** Write automatically-saving notes and keep track of various events and activities across your account.
* **Quick Transcriptions:** Ever encounter a plaque, letter, book, or other form of written media and would love to either read or save it for later? Now you can!
* **Note Sharing:** Share any note or transcript through custom chat links to any other field-notes user.
* **Pins Recognition:** If a note contains /way commands, field-notes will automatically generate pin links below the note window compatible with extended-pins and other waypoint addons!

## Installation

Download the latest release from the Releases page or use WowUp's `Install from URL` functionality for automatic updating.

## Usage

* Toggle visibility with the icon attached to the `Objective Tracker` frame

**Creating Notes:**
* Start writing or copy-paste from an alternate source. Notes autosave as you write but you have the option to hard-save as well.
* Click `Transcribe` above other forms of media to make a copy. (HTML support in the pipeline)

**Waypoint Support:**
* If a /way command is detected in a note, a corresponding pin link will instantly appear below the note. Click these to generate a waypoint with supported addons.
* Up to 10 waypoints per note are currently supported.

**Sharing Notes:**
* With a note open, clicking `Share` will send a link to your currently active chat channel. If another field-notes user clicks this, it will request the note from you and download a copy to their notes!
